# Math Quiz Game 🧮

An interactive console-based **Math Quiz** application developed in **C++**. This project challenges users with various mathematical operations across different difficulty levels, featuring automated question generation and real-time feedback.

## 🚀 Features
* **Dynamic Question Generation:** Automatically generates math problems based on the selected difficulty.
* **Multiple Difficulty Levels:** Supports Easy, Medium, Hard, and Mixed levels.
* **Varied Operations:** Includes Addition, Subtraction, Multiplication, Division, and Mixed operations.
* **Instant Feedback:** Provides immediate "Correct" or "Wrong" responses with the right answer shown for mistakes.
* **Result Summary:** Displays a final score including the number of right and wrong answers at the end of the quiz.

## 🛠️ Technical Overview
* [cite_start]**Language:** C++ (Standard: **C++20**)[cite: 14, 15].
* **Key Components:** Uses `structs` for managing quiz data and `enums` for difficulty and operation types.
* **Platform:** Windows Console (features screen color changes based on answers).

## 🎮 How to Play
1. **Launch the application.**
2. The game will present a series of math questions (Default: 5 questions).
3. Enter your answer for each question.
4. After completing the quiz, view your final performance summary to see if you passed.
